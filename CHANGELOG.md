# Changelog

## 0.1.0 (2025-01-29)

Initial open-source release.

- Workflow visualization for 17 LLM providers and 10 AI frameworks
- Interactive D3.js/Dagre graph with clickable nodes and source navigation
- AST-aware per-file caching (only changed files get reanalyzed)
- Instant local updates via tree-sitter (no LLM roundtrip on edit)
- Cross-file workflow detection with HTTP endpoint tracing
- Docker and manual backend self-hosting
- Copilot Chat integration (@codag participant)
